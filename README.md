# next-task-manager
タスクの進捗も確認できるタスク管理アプリ
## 実行方法
projectをpullして `npm run dev`
## 概要
ただ単純に**タスク管理アプリ**です。課題をすすめながら進捗を動かします
## 機能
タスクを追加、進捗と完了済みをいつでも変更可能、タスクはブラウザに保存されます。
## ページ構成
タスクのカードをクリックすると詳細、編集画面に行けます。
## 使用技術
* Nextjs
* Taiilwindcss
* shadcn/ui
* Node
* localStorage
## 工夫した点
できるだけ個人で使っていたReact版をうまく~~流用~~移植すること
